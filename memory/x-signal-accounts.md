## Value Tracking

**Scoring (updated after each scan):**
- **5** = Post led to actionable insight or build
- **4** = Learned something new and relevant
- **3** = Interesting but not actionable
- **2** = Noise, skippable
- **1** = Actively bad signal

**Monthly Reassessment:**
- Accounts averaging <3 → demote or drop
- Accounts averaging >4 → consider Tier 1 promotion
- New accounts discovered → trial in Tier 3
